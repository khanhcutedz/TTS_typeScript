# NODEJS PROJECT BASE

1. Clone
```bash
    git clone https://github.com/danghungtb26/nodejs_base.git
```
2. Install
```bash
    npm install
```
or
```bash
    yarn
```
3. Start
```bash
    npx run start
```
or
```bash
    yarn start
```
4. Check eslint
```bash
    npx run lint
```
or
```bash
    yarn lint
```
    

5. Check typescript

```bash
    npx run type
```
or
```bash
    yarn type
```
